## 第一步：安装工具

需要安装 Node.js 和 Wrangler（Cloudflare 官方上传工具）。

**安装 Node.js（如果没有）：**

**安装 Wrangler：**
```bash
npm install -g wrangler
```

## 第三步：上传文件

### 文件命名规范

每本书需要两个文件：
- `short.m4a` — 精华版（约 1~3 分钟）
- `long.m4a` — 完整版（约 15~30 分钟）

### 上传命令

```bash
# 格式：
wrangler r2 object put aireading-audio/zh/{书籍ID}/{文件名} --file {本地文件路径} --content-type audio/mp4 --remote

# 示例（上传 ID=4 刻意练习 的精华版）：
wrangler r2 object put aireading-audio/zh/4/short.m4a --file ./short.m4a --content-type audio/mp4 --remote

# 示例（上传 ID=4 刻意练习 的完整版）：
wrangler r2 object put aireading-audio/zh/4/long.m4a --file ./long.m4a --content-type audio/mp4 --remote
```

```
https://audio.aireading.com/zh/{书籍ID}/short.m4a
```

示例：
- https://audio.aireading.com/zh/4/short.m4a
