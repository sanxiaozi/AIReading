# 音频文件上传说明

## 文件命名规范

每本书需要两个文件：
- `short.m4a` —— 精华版（1~3 分钟）
- `long.m4a` —— 完整版（15~30 分钟）

## 上传路径规范

```
zh/{书籍ID}/short.m4a
zh/{书籍ID}/long.m4a
```

示例（人类简史 ID=3）：
```
zh/3/short.m4a
zh/3/long.m4a
```
