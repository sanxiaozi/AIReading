## 📋 音频质量Checklist

每个音频发布前检查:

- [ ] 无明显卡顿或中断
- [ ] 无异常噪音或杂音
- [ ] 发音准确，无明显错误
- [ ] 语速适中 (150-180 词/分钟)
- [ ] 响度标准化 (-16 LUFS)
- [ ] 文件大小合理 (<20MB/30min)
- [ ] 时长与文本匹配
- [ ] 章节标记正确 (如有)
- [ ] 元数据完整 (标题/作者)
- [ ] CDN可正常访问

## 🔧 常用命令

```bash
# 生成单本书音频
node scripts/generate-tts.mjs <bookId>

# 批量生成
node scripts/generate-tts.mjs 1 2 3 4 5

# 音频质检
ffprobe -v error -show_format audio.mp3

# 响度检测
ffmpeg -i audio.mp3 -af loudnorm=print_format=json -f null -

# 合并音频
ffmpeg -f concat -safe 0 -i files.txt -c copy output.mp3
```
