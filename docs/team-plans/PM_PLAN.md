# 🎯 产品经理 (PM) - 30天工作计划

## 角色定位
作为产品经理，我是项目的"总指挥"，负责确保团队朝着正确的方向前进，按时交付高质量的产品。

## 核心职责
- 产品规划与路线图管理
- 跨团队协调与沟通
- 进度追踪与风险管理
- 用户需求收集与优先级排序
- 数据分析与决策支持
