## 👤 阶段二：人工批量生成

### 准备工作
1. 登录 Google 账号
2. 打开 https://notebooklm.google.com
3. 准备好 `notebooklm-ready/` 文件夹

### 每本书操作流程（约 2 分钟）

```
步骤 1: 新建 Notebook
        → 点击 "New notebook"
        → 命名为书籍 ID（如 "03-人类简史"）

步骤 2: 上传内容
        → 点击 "Add source"
        → 选择 "Text" 或拖入 .txt 文件
        → 粘贴/上传对应书籍内容

步骤 3: 生成音频
        → 点击右侧 "Audio Overview"
        → 点击 "Generate"
        → 等待生成（约 3-5 分钟）

步骤 4: 下载音频
        → 生成完成后点击 "Download"
        → 保存到 downloads/ 文件夹
        → 命名格式：{book_id}_notebooklm.wav
```

### 批量操作技巧
- **多标签页并行**：同时开 5-10 个标签页，一边等待生成一边处理下一本
- **分批处理**：每次处理 10 本，休息一下
- **优先级顺序**：先处理 batch-1-priority

### 预计时间
| 批次 | 书籍数 | 预计时间 |
|------|--------|----------|
| Batch 1 (P1) | 4 本 | 8 分钟 |
| Batch 2 (P2) | 25 本 | 50 分钟 |
| Batch 3 (P3) | 21 本 | 42 分钟 |
| **总计** | **50 本** | **~100 分钟** |
