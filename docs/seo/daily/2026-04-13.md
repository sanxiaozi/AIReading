### 🔴 P0 CRITICAL 问题（持续第 14 天）

**问题**: sitemap 中存在 **104 个**无语言前缀的 URL，全部返回 **404**

**问题 URL 样例**：
```
https://aireading.com/about         → 404
https://aireading.com/faq           → 404  
https://aireading.com/category/business → 404
https://aireading.com/author/anders-ericsson → 404
...（共 104 个）
```

**根本原因**（已知 14 天，代码原封未动）：
```typescript
// 文件: src/app/sitemap.ts，第 37-38 行
function buildUrl(path: string, locale: Locale, baseUrl: string): string {
  if (locale === 'zh') {
    return `${baseUrl}${path}`;  // ← 🔴 BUG: zh版本未加 /zh 前缀
  }
  return `${baseUrl}/en${path}`; // ← en版本正确
}
```

**修复方案（5分钟，1行代码）**：
```typescript
// 将第38行改为：
return `${baseUrl}/zh${path}`;
```

**影响评估**：
- Google 每次抓取 sitemap 都遭遇 104 个 404 错误
- 持续 14 天将严重损害爬虫信任度
- 中文版内容页面在 Googlebot 视野中几乎不存在
- 每延误 1 天，SEO 恢复周期延长约 1 周
