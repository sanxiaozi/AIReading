### 🔴 P0 CRITICAL 问题（持续第 17 天）

**问题**: sitemap 中存在 **104 个**（除主页外）无语言前缀的 URL，均返回 **404**

> 注：sitemap 含 105 个无前缀 URL（含主页 `/`），主页本身可正常访问（200），其余 104 个全部 404

**问题 URL 按路径类型分布**：
| 类型 | 数量 | 示例 URL | 实际状态 |
|------|------|----------|---------|
| /about, /faq | 2 | `/about`，`/faq` | 404 🔴 |
| /category/* | 7 | `/category/business` | 404 🔴 |
| /author/* | 45 | `/author/cal-newport` | 404 🔴 |
| /books/* | 50 | `/books/1`... | ⚠️ 实际 200（书籍路由特殊） |

> ⚠️ **书籍页面说明**：sitemap 中含 `/books/*` 无前缀 URL，但实际访问 `/books/1` 等路径返回 200（因为书籍路由本身就在 `/books/[id]`，不区分语言前缀）。真实 404 的约 54 条非书籍 URL 和 50 条书籍的 zh 版 URL 指向错误位置。

**根本原因**（已知 17 天，代码原封未动）：
```typescript
// 文件: src/app/sitemap.ts，第 37-38 行
function buildUrl(path: string, locale: Locale, baseUrl: string): string {
  if (locale === 'zh') {
    return `${baseUrl}${path}`;  // ← 🔴 BUG: zh版本未加 /zh 前缀
  }
  return `${baseUrl}/en${path}`; // ← en版本正确
}
```

**修复方案（5分钟，1行代码）**：
```typescript
// 将第38行改为：
return `${baseUrl}/zh${path}`;
```

**累计损失评估**：
