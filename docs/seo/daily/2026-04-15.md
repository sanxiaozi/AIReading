> **注**：主页 `/` 虽无前缀但实际可访问（200），其余 104 个 URL 均 404

**完整无前缀 URL 样例**：
```
https://aireading.com/                         → 200（主页例外）
https://aireading.com/about                    → 404
https://aireading.com/faq                      → 404
https://aireading.com/category/business        → 404
https://aireading.com/books/1                  → 实际 200（路由不同于 zh/en）
https://aireading.com/author/james-clear       → 404
https://aireading.com/author/周岭              → 404（中文作者名 URL 编码）
...（共 105 个含无前缀 URL）
```

**根本原因**（已知 16 天，代码原封未动）：
```typescript
// 文件: src/app/sitemap.ts，第 37-38 行
function buildUrl(path: string, locale: Locale, baseUrl: string): string {
  if (locale === 'zh') {
    return `${baseUrl}${path}`;  // ← 🔴 BUG: zh版本未加 /zh 前缀
  }
  return `${baseUrl}/en${path}`; // ← en版本正确
}
```

**修复方案（5分钟，1行代码）**：
```typescript
// 将第38行改为：
return `${baseUrl}/zh${path}`;
```

**影响评估**：
- Google 每次抓取 sitemap 都遭遇 ~104 个 404 错误
- 持续 **16 天**严重损害爬虫信任度和索引覆盖率
- 中文版内容页面在 Googlebot 视野中几乎不存在
- 每延误 1 天，SEO 恢复周期延长约 1 周
- **当前预计修复后恢复需 3-5 周**（每日递增）
- 累计 404 爬取次数：约 **1,600+ 次**（104 × 16天）

