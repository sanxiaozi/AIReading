📘 AiReading – 全球多语言 AI 读书会项目文档（v1.0）

⸻

一、项目概要

1️⃣ 项目名称

AiReading

全球化 AI 读书与讲书平台 – 由 AI 生成多语言讲书内容，并支持用户与书互动学习。

⸻

2️⃣ 产品愿景（Vision）

“让每个人都能与知识对话。”
无论身处何地、使用何种语言，用户都能听懂一本书、提问一本书、
并将其智慧沉淀为个人成长资产。

⸻

3️⃣ 产品使命（Mission）

通过 AI 技术自动生成讲书稿与音频内容，
结合多语言支持与智能问答，让用户在 15 分钟内理解一本书的精华。
平台提供「精选内容 + 用户参与上传」的双轨内容生态，
成为全球最智能、最人性化的数字读书会。

⸻

二、产品定位

项目	描述
定位	AI版帆书 + NotebookLM式与书对话 + 全球多语言支持
目标人群	学习型成年人、学生群体、职场成长者、多语言学习者
核心价值	听懂、问懂、记住一本书
平台形态	Web + Android + iOS 全端一体化
内容来源	80% 官方精选（编辑部策划），20% 用户上传（UGC）
主要语言	中文、英文、泰文、印尼文（初期四语）


⸻

三、核心功能结构

📖 1. 精选讲书（Curation）
	•	官方编辑部定期精选优质书籍；
	•	AI 自动生成：
	•	3分钟精华讲书（Short Summary）
	•	10分钟深度讲书（Deep Summary）
	•	章节大纲（Outline）
	•	金句精选（Quotes）
	•	多语言讲述 + 语音播放（TTS）

⸻

💬 2. 与书对话（AI Chat）
	•	用户可在阅读或听书时随时提问；
	•	AI 检索原文（RAG），提供带引用的回答；
	•	多语言交互（中 / 英 / 泰 / 印尼）；
	•	支持「跨书提问」与“知识联想”。

⸻

🧠 3. 知识卡与笔记
	•	每次对话生成知识卡；
	•	自动总结读书笔记；
	•	用户可导出、合并、分享；
	•	后续版本支持知识图谱与成长曲线。

⸻

🎧 4. 听书与播放体验
	•	多音色、多语言 AI 朗读；
	•	播放器支持：倍速 / 下载 / 后台播放；
	•	支持章节跳转与播放记录；
	•	可在 Web / 移动端无缝衔接播放进度。

⸻

📤 5. 用户上传（UGC 弱化入口）
	•	用户可上传公共领域或自有版权书籍；
	•	AI 自动生成讲书稿与音频；
	•	默认私享，可申请公开发布；
	•	审核通过后出现在“社区精选”区。

⸻

👩‍💻 6. 编辑后台（CMS）
	•	书目管理（标题、语言、标签、版权状态）；
	•	内容生成流水线（文本→摘要→TTS）；
	•	专题/歌单管理（Collections）；
	•	首页版位配置（Placements）；
	•	审核与质量控制（AI评分+人工复核）；
	•	成本与运营监控（生成成本、播放指标、留存）。

五、系统架构

Frontend: Next.js + React Native (Expo)
Backend:  FastAPI / Node.js (REST + WebSocket)
Database: Postgres + pgvector
Storage:  MinIO / AWS S3 (音频与文本)
Cache/Queue: Redis / Celery
LLM: GPT-5 / Claude / Gemini (可调)
TTS: OpenAI / ElevenLabs / Azure
Translation: NLLB / DeepL
Auth: OAuth + JWT
Infra: Docker Compose / Kubernetes / Vercel + Fly.io

核心流程：
Text → Segmentation → Embedding → Summarization → TTS → Publish
